# Circuit Core
Circuit Core is the core mod for most of AussieBox's mods. Currently, it includes:
- Advancement helper methods
- Block, Item, and BlockItem helper methods
- Automated ItemGroup registration
- PlayerAnimationLib utilities

... with much more planned soon! It also currently supports four versions: ```1.21.1```, ```1.21.8```, ```1.21.10```, and ```1.21.11```.
